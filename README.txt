재현완성창 앱

1. GitHub Pages 앱 링크를 PC용 Chrome 또는 Edge로 엽니다.
2. '저장 목록' → '저장 폴더 선택'을 누르고 고객 견적을 보관할 PC 폴더를 지정합니다.
3. 최초 선택 후에는 견적 작성 내용이 브라우저와 선택한 PC 폴더에 자동 저장됩니다.
4. 브라우저를 다시 실행했을 때 권한 확인이 표시되면 '폴더 권한 다시 허용'을 누릅니다.
5. 폴더에는 견적별 JSON 파일이 저장되며, '폴더 새로고침'으로 다시 불러올 수 있습니다.
6. 고객 정보는 GitHub에 업로드되지 않습니다. GitHub에는 앱 프로그램과 기본 단가만 배포됩니다.
7. 모바일 브라우저는 PC 폴더 연결 기능이 제한될 수 있어 '현재 견적 내보내기/가져오기'를 사용합니다.
8. '견적서 출력'에서 인쇄 또는 PDF 저장이 가능합니다.
