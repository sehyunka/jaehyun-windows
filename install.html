앱처럼 설치하기

Google Play 없이 링크만으로 설치해 홈 화면에서 바로 실행할 수 있습니다.

앱 바로 열기
1

Android / Chrome

앱 설치하기를 누르고 설치창에서 설치를 선택하세요. 설치창이 안 뜨면 브라우저 메뉴의 앱 설치 또는 홈 화면에 추가를 선택하세요.

2

iPhone / iPad

Safari에서 이 페이지를 열고 공유 버튼을 누른 뒤 홈 화면에 추가 → 추가를 선택하세요.

3

설치 후 사용

홈 화면의 재현견적 아이콘을 누르면 주소창 없이 앱처럼 열립니다. 견적 데이터는 선택한 PC 폴더 또는 현재 브라우저에 저장됩니다.

소수 사용자에게는 이 설치 페이지 링크를 문자·카톡으로 보내면 됩니다.

버튼을 눌러도 설치창이 안 뜰 때
  1. Android Chrome에서 이 페이지를 다시 엽니다.
  2. 오른쪽 위 점 3개 메뉴를 누릅니다.
  3. 앱 설치 또는 홈 화면에 추가를 선택합니다.
  4. 홈 화면에 생긴 재현견적 아이콘으로 실행합니다.

Android 설치 방법

설치창이 자동으로 뜨지 않으면 아래 순서대로 설치해 주세요.

  1. Chrome 오른쪽 위 점 3개 메뉴를 누르세요.
  2. 앱 설치 또는 홈 화면에 추가를 선택하세요.
  3. 확인창에서 설치 또는 추가를 누르세요.
  4. 홈 화면의 재현견적 아이콘으로 실행하세요.
